# Görev Akışı

Ekip görev takip sitesi. `index.html` dosyasına çift tıklayarak aç.

## Sayfalar

| Dosya | Açıklama |
|-------|----------|
| `index.html` | Giriş |
| `leader.html` | Lider paneli — proje ve kullanıcı yönetimi |
| `projects.html` | Proje listesi |
| `project.html` | Proje detayı ve ekip |
| `tasks.html` | Görevler |
| `kanban.html` | Kanban panosu |
| `my-tasks.html` | Kişisel görevler |
| `dashboard.html` | Admin paneli |
| `users.html` | Kullanıcı yönetimi |
| `profile.html` | Profil |

Veriler tarayıcıda (`localStorage`) saklanır.
