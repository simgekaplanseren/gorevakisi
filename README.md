# TaskFlow

TaskFlow, ekiplerin projelerini ve görevlerini yönetebildiği bir web uygulamasıdır. Bu proje, staj için güçlü bir proje fikri olarak hem gerçek hayat kullanım senaryolarını kapsar hem de temel proje yönetimi özelliklerini içerir.

## Amaç

- Proje oluşturma
- Görev ekleme
- Görevleri düzenleme
- Görev ilerlemesini takip etme

## Kullanıcı roller

### Admin
- Proje oluşturabilir
- Görev oluşturabilir
- Kullanıcı ekleyebilir

### Kullanıcı
- Kendisine atanmış görevleri görebilir
- Görev durumunu güncelleyebilir
- Göreve yorum ekleyebilir

## Teknoloji yığını

- Backend: ASP.NET Core Web API (C#)
- Frontend: React + Vite
- Database: SQL Server
- Versiyon Kontrol: Git + GitHub
- IDE: Cursor

## İlk sürümde yer alacak temel özellikler

- ✅ Kullanıcı kayıt olma
- ✅ Giriş yapma
- ✅ Proje oluşturma
- ✅ Görev oluşturma
- ✅ Görev düzenleme
- ✅ Görev silme
- ✅ Görev durumunu değiştirme

## Notlar

Bu proje, staj başvuruları için gerekli temel yetkinlikleri göstermek amacıyla sade ama işlevsel bir MVP olarak tasarlanmıştır. İlerleyen sürümlerde rol yönetimi, proje panosu, bildirimler ve raporlama gibi özellikler eklenebilir.
